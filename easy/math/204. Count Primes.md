### 1 Brute Force
```java
public int countPrimes(int n) {
    int count = 0;
    for (int i = 2; i < n; i++) {
        if (isPrime(i))
            count ++;
    }
    return count;
}

private boolean isPrime(int k) {
    for (int i = 2; i < k; i++) {
        if (k % i == 0)
            return false;
    }
    return true;
}
```


### 2 a little optimization

```java
public int countPrimes(int n) {
    int count = 0;
    for (int i = 2; i < n; i++) {
        if (isPrime(i))
            count ++;
    }
    return count;
}

private boolean isPrime(int k) {
    int sqrt = (int)Math.sqrt(k);
    for (int i = 2; i <= sqrt; i++) {
        if (k % i == 0)
            return false;
    }

    return true;
}
```

### 3 use an array for filtering
```java
public int countPrimes(int n) {
    boolean[] toFilter = new boolean[n]; 
    // == true means that's not a prime and we need to filter it out.

    int counter = 0;
    int sqrt = (int) Math.sqrt(n);

    for (int i = 2; i < n; i++) {
        if (!toFilter[i]) {
            counter++;

            if (i > sqrt) continue;

            for (int j = i; j * i < n; j++) {
                toFilter[j * i] = true;
            }
        }
    }

    return counter;
}
```
